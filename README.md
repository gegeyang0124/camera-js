# camera-js
使用纯js实现拍照，可运行于各种浏览器，特别是手机端（正在开发封装中）

### 使用 引入camera.js（方法参数，请查看源文件）
```html
<script src="./.../camera-js/camera.js"></script>
<script>
Camera.takePicture();//拍照
Camera.takePictureSilent();//静默拍照
</script>
```
